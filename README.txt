九星気学 四盤暦 年盤・月盤テキスト出力ツール

■ ファイル
- index.html

■ 使い方
1. index.html をブラウザで開く
2. 対象年を入力（2021〜2030）
3. 必要なら本命星を選ぶ
4. 「生成する」を押す
5. 「テキストをコピー」でコピペ用テキストを取得

■ GitHub Pages 公開手順
1. GitHubで新しいリポジトリを作る
2. index.html をアップロードする
3. リポジトリの Settings → Pages を開く
4. Deploy from a branch を選ぶ
5. Branch を main / root にして Save
6. 数分待つと公開URLが発行される

■ 備考
- 年盤は立春基準
- 月盤は 2月節〜翌1月節の12節月
- 2021〜2030のデータを内蔵
